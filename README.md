# momentum

크롬 앱에 있는 [momentum](https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca?hl=ko)을 기반으로 Vanilla JS를 사용하여서 만들었다.

## Features

- [x] Clock
- [x] To Do List
- [x] Weather
- [ ] Daily Quote
- [x] Daily Photo
- [x] Offine Support
